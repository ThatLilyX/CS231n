# Assignment 1: Image Classification

## Overview
This assignment covers the foundations of image classification including:
- k-Nearest Neighbor (kNN) classifier
- Support Vector Machine (SVM)
- Softmax classifier
- Two-layer neural network
- Image feature extraction

## Files
- `knn.py` - k-Nearest Neighbor implementation
- `linear_svm.py` - Linear SVM classifier
- `softmax.py` - Softmax classifier
- `two_layer_net.py` - Two-layer neural network
- `features.py` - Image feature extraction (HOG, color histogram)

## Instructions
1. Download the CIFAR-10 dataset (will be auto-downloaded on first run)
2. Implement the classifiers in the respective files
3. Run the Jupyter notebooks to test your implementations
4. Answer the inline questions in the notebooks

## Dataset
CIFAR-10: 60,000 32x32 color images in 10 classes
