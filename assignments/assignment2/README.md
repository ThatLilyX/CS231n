# Assignment 2: Fully Connected and Convolutional Networks

## Overview
This assignment covers:
- Fully-connected neural networks
- Batch normalization
- Dropout regularization
- Convolutional neural networks
- PyTorch/TensorFlow framework introduction

## Files
- `fc_net.py` - Fully-connected neural network with arbitrary depth
- `batch_norm.py` - Batch normalization layer
- `dropout.py` - Dropout layer implementation
- `cnn.py` - Convolutional neural network
- `optim.py` - Optimization algorithms (SGD, Adam, RMSprop)
- `layers.py` - Forward and backward pass for various layers

## Key Concepts
- Backpropagation through fully-connected layers
- Batch normalization for training stability
- Dropout for preventing overfitting
- Convolutional layers for spatial feature learning
- Modern optimization techniques

## Instructions
1. Implement the modular layer functions (forward/backward)
2. Build fully-connected networks with arbitrary depth
3. Add batch normalization and dropout
4. Implement convolutional networks
5. Experiment with different architectures and hyperparameters
